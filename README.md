# NotesAura AI Chatbot (Node.js)

AI-powered chatbot built with Node.js, Express, and Sarvam AI.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file:
```
SARVAM_API_KEY=your_api_key_here
PORT=3000
```

3. Run server:
```bash
npm start
```

4. Open browser: http://localhost:3000

## Features
- Real-time AI chat with Sarvam AI
- Markdown & code syntax highlighting
- Copy code & table functionality
- Fully responsive design
- NotesAura app context integration
